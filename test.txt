git  init :初始化一个版本库.
添加文件到版本库:
1.  git add test.txt 
2.  git commit -m "修改说明"
3.  git status :查看工作区的状况.
4.  git diff :查看文件修改内容.
5.  git log:从近到远查看提交日志
6.  git --pretty=oneline:所有信息显示在同一行
7.  log中的一大窜数字是commit  id(版本号).
8.  git reset --hard HEAD^ : 当前版本回退到上一版本
